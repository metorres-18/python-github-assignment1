#Study Time Tracker

This is a simple Python program that helps you track your weekly study time and if you have reached you weekly goal. You enter how many hours you studied today, and the program projects how many more hours you will study to reach your goal.

#How to run

Make sure that you have python is installed.

Sample Output:

Entered hours studied: 4
You need 6.0 more hours to reach a 10-hour weekly goal.
